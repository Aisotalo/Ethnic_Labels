This is a repository for work done on Discourse Analysis of Ferguson using The Washington Post, The New York Times and The Wall Street Journal as a corpus. 60 articles were generated per publication, within 2 date ranges: the first 30 articles from August 9th 2014, and the first 30 articles from November 24th 2014. A total number of 180 articles exists within the corpus.

See: 

/Ethnic_Labels/doclist.txt for a breakdown and basic description of the subdirectory contents
/Ethnic_Labels/commands.txt for a log of some common commands used during the pre-processing step
/Ethnic_Labels/Total_Raw_Tokens.txt for a total count of target tokens within the corpus
